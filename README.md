Кейс-задача № 3. Синергия
Сначала я нашел в массиве индексы максимального и минимального элементов (использовал функции max() и min(), затем метод index()). Определил левую и правую границы промежутка между ними (исключая сами максимум и минимум) - взял меньший индекс + 1 и больший индекс - 1. Проверил, есть ли между ними элементы: если левая граница больше правой - значит элементов нет, сумма равна 0. Прошел по этой части массива в цикле и сложил только те числа, которые меньше нуля.
